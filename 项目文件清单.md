## 八、如何使用

### 快速启动
```bash
# 1. 启动依赖服务
docker-compose up -d

# 2. 启动应用
mvn spring-boot:run

# 3. 访问API文档
http://localhost:8080/doc.html
```

### 测试API
```bash
# 上传文件
curl -X POST http://localhost:8080/api/files/upload \
  -F "file=@test.pdf"

# 获取文件列表
curl http://localhost:8080/api/files

# 下载文件
curl -O http://localhost:8080/api/files/1/download
```

详细说明请查看 **[启动指南.md](./启动指南.md)**
